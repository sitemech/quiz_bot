# NestQuiz - Telegram бот для уведомлений о квизах в Уфе

## 📋 Описание

NestQuiz — это Telegram-бот, который автоматически отслеживает расписание квизов на сайте quizplease.ru для города Уфа и отправляет подписчикам уведомления о ближайших играх. Бот фильтрует квизы для новичков и отправляет ежедневные напоминания.

## ✨ Основной функционал

- 🔍 **Автоматический парсинг расписания** с сайта quizplease.ru
- 📅 **Ежедневные уведомления** о квизах на сегодня и завтра
- 🎯 **Фильтрация квизов** (исключает игры для новичков)
- 👥 **Управление подписками** с сохранением в JSON
- ⏸️ **Управление уведомлениями** (пауза/возобновление)
- 🔄 **Режим отладки** для тестирования

## 🚀 Быстрый старт

### Требования

- Python 3.7+
- Telegram Bot Token (получить у [@BotFather](https://t.me/BotFather))

### Установка

1. Клонируйте репозиторий или скачайте файлы проекта

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Настройте конфигурацию:
   - Откройте `config.py`
   - Установите токен бота одним из способов:
     - Через переменную окружения: `export BOT_TOKEN="ваш_токен"`
     - Или измените значение по умолчанию в `config.py`

4. Запустите бота:
```bash
python main2.py
```

## 📖 Команды бота

| Команда | Описание |
|---------|----------|
| `/start` или `/subscribe` | Подписаться на уведомления и получить информацию о ближайшем квизе |
| `/next` | Показать следующий ближайший квиз |
| `/pause` | Приостановить отправку уведомлений |
| `/resume` | Возобновить отправку уведомлений |

### Основные компоненты

#### 1. Парсинг расписания (`fetch_quiz_schedule()`)
- Запрашивает HTML-страницу с расписанием
- Парсит данные с помощью BeautifulSoup
- Извлекает: дату, название, место проведения, время, описание
- Возвращает список кортежей с информацией о квизах

#### 2. Управление подписчиками
- `load_subscribers()` - загрузка из JSON при старте
- `save_subscribers()` - сохранение при изменении списка
- Хранение в множестве `CHAT_IDS` для быстрого доступа

#### 3. Планировщик уведомлений
- **Режим продакшн**: ежедневно в 05:00 UTC
- **Режим отладки**: каждые 10 секунд (для тестирования)
- Фильтрация квизов на сегодня и завтра
- Исключение квизов для новичков

#### 4. Обработчики команд Telegram
- Асинхронная обработка через `pyTelegramBotAPI`
- Логирование всех входящих сообщений
- Обработка ошибок с информативными сообщениями

## ⚙️ Конфигурация

### Переменные в `main2.py`

```python
DEBUG = False  # Режим отладки (True = уведомления каждые 10 сек)
DEBUG_DATE = None  # Подмена текущей даты для тестирования
NOTIFICATIONS_ENABLED = True  # Глобальный флаг уведомлений
```

### Настройка времени уведомлений

В функции `scheduler_production()` измените время:
```python
schedule.every().day.at("05:00").do(send_daily_notification)  # UTC время
```

### URL расписания

URL настраивается в переменной `URL`:
```python
URL = "https://ufa.quizplease.ru/schedule?QpGameSearch%5BcityId%5D=6&..."
```

Для другого города измените параметр `cityId` в URL.

## 🔧 Разработка

### Режим отладки

Для тестирования включите режим отладки:
```python
DEBUG = True
```

В этом режиме:
- Уведомления отправляются каждые 10 секунд
- Можно использовать `DEBUG_DATE` для тестирования с конкретной датой

### Логирование

Бот использует стандартный модуль `logging`:
- Уровень: `INFO`
- Формат: `%(asctime)s - %(levelname)s - %(message)s`
- Все операции логируются в консоль

### Обработка ошибок

- Парсинг: проверка наличия элементов перед извлечением
- HTTP-запросы: таймаут 15 секунд, обработка статус-кодов
- Telegram API: обработка исключений при отправке сообщений
- Файлы: безопасная работа с JSON

## 🚢 Развертывание

### Локальный запуск

```bash
python main2.py
```

### Запуск на сервере (systemd)

Создайте файл `/etc/systemd/system/nestquiz.service`:

```ini
[Unit]
Description=NestQuiz Telegram Bot
After=network.target

[Service]
Type=simple
User=ubuntu
WorkingDirectory=/path/to/NestQuiz
Environment="BOT_TOKEN=your_token_here"
ExecStart=/usr/bin/python3 /path/to/NestQuiz/main2.py
Restart=always
RestartSec=10

[Install]
WantedBy=multi-user.target
```

Запуск:
```bash
sudo systemctl enable nestquiz
sudo systemctl start nestquiz
sudo systemctl status nestquiz
```

### Запуск в фоне (screen/tmux)

```bash
screen -S nestquiz
python main2.py
# Ctrl+A, затем D для отсоединения
```

## 🐛 Известные проблемы и ограничения

1. **Парсинг дат**: Используется фиксированный год (2025) в некоторых версиях. В `main2.py` это исправлено.
2. **Защита от ботов**: Сайт может блокировать запросы. В `main2.py` добавлен User-Agent.
3. **Структура HTML**: При изменении структуры сайта может потребоваться обновление парсера.
